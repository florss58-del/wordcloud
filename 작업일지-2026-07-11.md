# 작업일지 — 2026-07-11

남아 있던 TODO 중 **결과 보기 + CSV 내보내기**를 구현. 발표 화면에서 수업 직후 단어 빈도를 확인하고 엑셀로 내려받는 흐름.

## 구현 내용

- `present.html` 툴바에 **"결과 보기"** 버튼 추가 (PNG 저장 왼쪽)
- 결과 패널(`#resultsOverlay`): 카드형 오버레이
  - 머리말: 질문 + 날짜·코드·참여 인원·제출 수·단어 종류 수
  - 표: 순위 · 단어 · 비율 막대 · 횟수(비율 %) — 빈도 내림차순, 동률이면 가나다순
  - 발표 화면에서 숨긴 단어는 기본 제외. "숨긴 단어 포함" 체크 시 취소선 표시로 함께 집계
  - 빈 상태: "아직 집계할 단어가 없습니다"
  - 닫기: 버튼 · 바깥 클릭 · Esc
- **CSV 내보내기**: 순위·단어·횟수·비율(%)·표시(표시/숨김) 5열. 엑셀 한글 깨짐 방지를 위해 UTF-8 BOM(`﻿`) + CRLF, 큰따옴표 이스케이프. 파일명 `wordcloud_{코드}_{날짜}.csv`
- `todayStamp()` 헬퍼 추가 — PNG 저장의 파일명이 `toISOString()`(UTC)을 쓰던 것을 로컬 날짜로 통일(저녁 수업에서 하루 밀리는 문제 방지)
- `css/style.css`: `.results-*` 스타일(Apple 폴리시드 톤, 다크 대응, 모바일에서 막대 열 숨김). 캐시 버전 `?v=8`로 통일
